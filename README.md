# -
一款基于HTML的报价单生成器


网页核心功能
该工具是一款面向产品报价场景的报价单生成器 ，核心功能围绕报价单的制作、管理与多模版导出展开，具体包括：
1、 分组与产品管理
A、分组操作：支持新增分组、自定义分组名称、删除分组，可通过拖拽调整分组顺序，满足报价单的模块化分类需求。
B、产品行管理：
新增 / 删除产品行，支持拖拽调整产品行顺序，可视化拖动指引（高亮线提示）提升操作直观性；
C、产品信息一键引出：涵盖设备名称、品牌、型号、单价、数量、单位、备注、技术参数等维度，支持自定义输入或从内置参数库选择型号一键引出对应参数。
D、型号选择与参数管理：
弹窗式设备选型界面，支持按一级 / 二级分类筛选、关键词搜索（可指定搜索范围：名称 / 品牌 / 型号或技术参数）；备注 / 技术参数支持弹窗式编辑，适配大段文本输入，提升编辑体验。
E、内置渠道产品参数库，支持自定义设备


2、 数据计算与导出
自动合计：实时计算所有设备的金额合计，直观展示总报价。
多模板报价单导出。

3、工作内容可保存
可导出当前配置的产品清单（JSON 格式），后续需要修改时，只需重新导入该文件，即可恢复之前的工作进度，满足后续报价单修改需求。
